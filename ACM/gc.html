ACM入门—介绍有关ACM-ICPC的常识

介绍ACM-ICPC的入门知识，引导参加ACM-ICPC活动。

ACM程序设计大赛

    ACM程序设计大赛（ACM International Collegiate Programming Contest, 简称ACM-ICPC或ICPC）是大学级别最高的脑力竞赛，素来被冠以"程序设计的奥林匹克"的尊称。
    大赛至今已有近40年的历史，是世界范围内历史最悠久、规模最大的程序设计竞赛。
    比赛形式是：从各大洲区域预赛出线的参赛队伍，于指定的时间、地点参加世界级的决赛，由1个教练、3个成员组成的小组应用一台计算机解决7到13个生活中的实际问题。

2015年比赛计算机环境

  • 操作系统: Linux - Ubuntu 15.10 (64位)
  • 编程语言:
    • C/C++: gcc/g++ 5.2.1
    • Java : openjdk 1.8.0_66
    • Scala 2.11.6
    • Python 2.7.10 , 3.4.3
  • 编程环境:
    • code::blocks 13.12
    • eclipse 4.5.1 with CDT, scala-ide, pydev
    • netbeans IDE 8.0.2
    • vi/vim/gvim 7.4.712
    • emacs 24.5.1
    • gedit 3.10.4
    • geany 1.25
  • 评判系统: pc2-9.2.3

必要的知识与能力

  • 计算机语言：C、C++和Java，至少掌握前两种语言
  • 程序设计：结构化程序设计、自顶向下逐步细化和面向对象程序设计
  • 数据结构：系统地了解和掌握
  • 算法设计与分析：系统地了解和掌握
  • 逻辑：较强的逻辑推理能力，能够解决一般的智力问题
  • 数学：坚实的数学基础，较强的思维与计算能力
